# PieChart - Vuejs

> A PieChart with Vuejs

![Demo](https://cdn-images-1.medium.com/max/1600/1*DRHLLxye1Nh2aTDo8v_Mxg.gif "PieChart")

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
